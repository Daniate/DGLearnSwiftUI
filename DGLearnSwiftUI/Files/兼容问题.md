#  兼容问题

为了让低版本的`Xcode`可以正常编译使用了高版本`API`的项目，可使用如下写法：

```swift
#if swift(>=5.5)
    if #available(iOS 15.0, *) {
    
    }
#endif

#if swift(>=5.3)
    if #available(iOS 14.0, *) {
    
    }
#endif
```
